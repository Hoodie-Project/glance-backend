# glance-backend

Spring Boot backend for Glance.

## Requirements

- Java 21
- PostgreSQL

## Getting Started

```bash
# Run with local profile
./gradlew bootRun --args='--spring.profiles.active=local'

# Run tests
./gradlew test
```

## Tech Stack

- Java 21
- Spring Boot 4.0
- Gradle (Kotlin DSL)
- PostgreSQL
- Spring Security
