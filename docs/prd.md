# PRD (Product Requirements Document)

## 서비스 개요

**Glance**는 위치 기반 익명 커뮤니티 앱이다. 사용자가 현재 위치에서 스레드(게시글)를 작성하면 주변 사람들이 지도나 피드에서 확인할 수 있다. 회원가입 없이 비밀번호만으로 글을 관리한다.

## 핵심 기능

### 스레드 작성
- 위치(위도/경도), 닉네임, 제목, 내용, 성별, 해시태그, 동물상, 분위기 스타일 입력
- 비밀번호 미입력 시 서버가 자동 생성 후 반환
- 작성 시 네이버 Reverse Geocoding으로 지역 정보 자동 저장

### 피드
- 전체 피드: 최신순 cursor 기반 무한스크롤
- 주변 피드: 반경(0.5km / 2km / 5km) 내 스레드 무한스크롤
- 태그 검색: 해시태그 기반 전역 검색

### 지도
- 줌인: 현재 뷰포트 내 스레드 핀 표시 (범위 제한 ±0.072°)
- 줌아웃: 행정동 단위 마커 및 스레드 수 표시

### 댓글
- 스레드에 댓글 작성 (닉네임, 내용, 비밀번호)
- 오래된 순 정렬

### 좋아요
- 스레드/댓글 좋아요 토글
- IP당 1회 제한

### 삭제
- 스레드: 비밀번호 검증 후 소프트 삭제 (복구 가능)
- 댓글: 비밀번호 검증 후 하드 삭제

## 비즈니스 규칙

| 규칙 | 내용 |
|------|------|
| 레이트 리밋 | 스레드 생성 IP당 1분 최대 3건 |
| 비밀번호 자동 생성 | 8자, 혼동 문자(0, 1, I, L, O) 제외 |
| 비밀번호 길이 | 사용자 입력 시 4~8자 |
| 소프트 삭제 | 스레드만 해당, 어드민에서 복구 가능 |
| 좋아요 중복 방지 | IP + 대상 ID 기준 DB Unique Constraint |
| 지도 핀 범위 | 위도/경도 각 0.072° 초과 시 400 에러 |

## 콘텐츠 제한

| 항목 | 제한 |
|------|------|
| 스레드 제목 | 최대 30자 |
| 스레드 내용 | 최대 500자 |
| 댓글 내용 | 최대 300자 |
| 닉네임 | 최대 20자 |
| 해시태그 | 최대 5개 |

## 관리자 기능

Thymeleaf 기반 웹 어드민 (`/admin/`)

- 대시보드: 스레드/댓글/지역 통계
- 스레드 목록: 키워드 검색, 삭제된 글 포함 필터
- 스레드 상세: IP/기기 정보 확인, 강제 삭제, 복구
- 댓글 목록: 삭제
- 지역 목록 확인
